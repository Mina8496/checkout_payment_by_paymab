# Checkout Payment App using Paymob 💳

A Flutter application demonstrating payment integration using Paymob Payment Gateway with a clean, scalable architecture and Bloc/Cubit state management.

---

## 🚀 Features

* Secure payment processing with Paymob
* Clean Architecture implementation
* Bloc/Cubit State Management
* API Integration
* Error Handling
* Loading & Success States
* Responsive UI
* WebView Payment Flow

---

## 🛠 Tech Stack

* Flutter
* Dart
* Paymob Payment Gateway
* Bloc / Cubit
* REST API
* Clean Architecture

## ⚙️ Getting Started

Clone the repository:

```bash
git clone https://github.com/Mina8496/checkout_payment_by_paymab.git
```

Navigate to the project:

```bash
cd checkout_payment_by_paymob
```

Install dependencies:

```bash
flutter pub get
```

Run the application:

```bash
flutter run
```

---

## 🔑 Paymob Configuration

Configure your Paymob credentials before running the application:

```dart
const String apiKey = "YOUR_API_KEY";
const int integrationId = YOUR_INTEGRATION_ID;
```

⚠️ Never commit production API keys or sensitive credentials to public repositories.
